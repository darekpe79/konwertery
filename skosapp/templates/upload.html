Upload Excel File for SKOS Conversion

Instructions:

  1. Excel File: Upload your Excel file with the SKOS data using the provided field.
  2. Mapping Columns: The form will dynamically generate fields for each SKOS property based on your Excel file. Enter the column names from the Excel sheet that correspond to each SKOS property. If a SKOS property does not have a corresponding column in your Excel file, leave that field empty.
  3. Handling Multilingual Data: For columns containing multiple values or multilingual labels, separate each value with a comma. Attach a language tag to a label by appending "@" to the label. For example: "Label @en, Etiqueta @es".
  4. URI for Concepts: Make sure to fill out the field corresponding to 'skos:concept' as it determines the URI for each concept in the RDF output.
  5. Output Format: Choose the desired RDF serialization format for the output file from the 'Output Format' dropdown menu.
  6. Generating RDF: After completing the form, submit it to convert your Excel data into an RDF file in the selected format. The RDF file will be available for download once the conversion is complete.

This tool is designed to facilitate the creation of SKOS-based RDF files from Excel data, simplifying semantic data management. It is suitable for both experienced semantic web developers and newcomers. Try it out to enhance your data management workflow.
